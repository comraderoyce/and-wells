Notes on how I'm using ```tmux```.

<!--more-->

Install on OSX
: ```$ brew install tmux``` 

Launch tmux attached to command
: ```$ tmux -c irssi```

Add new window to session
: ```[mod] c```

Slipt pane vertically
: ```[mod] %```

Slipt pane horizontally
: ```[mod] "```

Resize pane
: ```:resize-pane -L 50```
